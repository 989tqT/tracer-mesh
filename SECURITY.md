# Security Policy

## Supported Versions

We actively support and patch security vulnerabilities in the following versions:

| Version | Supported          |
| ------- | ------------------ |
| v0.1.x  | :white_check_mark: |
| < v0.1  | :x:                |

## Reporting a Vulnerability

If you identify a security vulnerability in Tracer Mesh, please do **not** open a public issue. We adhere to a responsible disclosure policy to protect users and systems deploying our software.
